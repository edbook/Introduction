2. Features¶

2.1. Extensions¶

2.1.1. Mathematical symbols¶

Mathematical symbols are written using a LaTeX syntax and display using MathJax. Either inline \(e^{\cos(x)}\) or as equations

\[\frac{d}{dz} \cos(z) = \sum_{n\in \mathbb N} \frac {d}{dz}\, \frac{(-1)^n}{(2n)!} z^{2n} = \sum_{n\geq 1} \frac{(-1)^n}{(2n-1)!}z^{2n-1} = -\sin(z)\]

2.1.2. Links¶

It is possible to link to every section, which is useful when making solutions, in emails and on Piazza.

2.1.3. Geogebra¶

Geogebra (http://geogebra.org) is an excellent program to make interactive (and good looking) applets which can be embedded.



2.1.4. Sage¶

Sage can be used to put in applets written in Sage, Python, Octave, R, etc. The code can be hidden or shown, which then enables students to play with it.



2.1.5. Panopto¶

Videos from Youtube or Panopto can be embedded.



2.2. Other¶

  • A pdf-version is available who those which to print them out.

  • Index.

  • (Icelandic) translations of mathematical can be shown, for example heiltalaen: integer
    Smelltu fyrir ítarlegri þýðingu.    .

  • Translated word are collected in a dictionary. This feature uses the dictionary of the Icelandic Mathematical Society (http://stae.is/os).

  • It is possible to collect statistics through the web server, Google Analytics and Panopto.

This can be used to see how students use the notes. If there are some parts they use more than others and if there are some parts (e.g. proofs) which they skip.
